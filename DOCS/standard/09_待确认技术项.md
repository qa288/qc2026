# 09 待确认技术项

> 本文档只保留仍可能影响联调或上线的事项。已经拍板的内容不再重复讨论。

## 1. 已确认项

- 正式域名：`qc.tyos.cc`
- 部署方式：Docker Compose
- 技术栈：FastAPI + PostgreSQL + Redis + Celery + 独立 scheduler
- 第一阶段只做阈值告警
- 通知先做配置结构与适配层，实际发送可后置开启
- 账户汇总失败时接受计划聚合降级
- Token 策略采用单点刷新、共享最新 token
- 后台可公网开放，但保留登录和审计
- 第一阶段做公开页，不做大屏
