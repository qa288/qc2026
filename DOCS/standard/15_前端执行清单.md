## 1. 页面范围

前端第一阶段只做：

- `/login`
- `/`

主看板包含：

- 总览
- 账户
- 经营拆解
- 计划
- 素材
- 关键词归属
- 账号与权限
- 预警规则

## 2. 必须支持

- 排序
- 搜索
- 日 / 昨日 / 近7天 / 近30天 / 指定日期范围
- 详情侧栏
- 自动刷新后保留排序和筛选

## 3. 不做

- 大屏页
- 定时报表配置

## 4. 对接原则

- 只调用内部 API
- 不暴露 Token、官方字段、OpenClaw 配置
- 空值显示 `--`
