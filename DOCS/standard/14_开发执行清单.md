# 14 开发执行清单

本文档是开发总表。  
固定决策一律引用 [13_已确认决策基线.md](/Users/xy/千川/DOCS/standard/13_已确认决策基线.md)，不再重复抄写。

## 1. P0 必做

- 搭建 FastAPI + PostgreSQL + Redis + Celery + Scheduler
- 登录、会话、角色、权限
- Token 单点刷新与共享
- 官方接口接入
- 账户汇总降级
- 归属人 / 关键词 / 人工修正模型
- 余额 / 共享钱包 / 消耗 / 爆单预警骨架

## 2. P1 必做

- 总览、账户、经营拆解、计划、素材、归属、权限、预警规则页面
- 排序、筛选、`日 / 昨日 / 近7天 / 近30天 / 指定日期范围`
- 阈值告警
- 通知渠道配置结构

## 3. P2 必做

- 健康检查
- 日志
- 备份
- 上线与回滚

## 4. 开发要求

- 不再使用旧 Node 栈
- 不实现大屏页 / 定时简报
- 页面只查内部 API
- 其它服务器读取 token-latest，不自行刷新
