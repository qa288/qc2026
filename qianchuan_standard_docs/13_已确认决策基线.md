# 13 已确认决策基线

最后更新：`2026-03-22`

后续开发、联调、部署如与本文档冲突，以本文档为准。

## 1. 产品与路由

- 产品定位：公司内部使用的经营与投流决策平台
- 路由：
  - `/login`
  - `/`
  - `/api/*`
- 第一阶段不做公开页、大屏页、独立 `/admin`

## 2. 部署与域名

- 正式域名：`ss.tyos.cc`
- 部署方式：Linux 单机 + Docker Compose
- 服务器已安装 `1Panel`，作为反向代理、站点管理和 Docker 运维入口
- 后台可公网开放，但保留登录和审计

## 3. 技术栈

- Web/API：FastAPI
- 主数据库：PostgreSQL
- 缓存 / 队列 / 锁：Redis
- 异步任务：Celery
- 定时调度：独立 scheduler
- 前端：轻前端

## 4. 通知

- 第一阶段只做阈值告警
- 通道先走 OpenClaw -> Feishu
- 后续再切 DingTalk

## 5. 账户汇总

- 允许官方账户汇总失败时自动降级为计划聚合
- 页面和导出必须保留来源标记

## 6. Token

- 单点刷新，共享最新 token
- PostgreSQL 为主真源
- `/app/data/qianchuan_latest_token.json` 为缓存副本
- Redis 负责刷新锁
- 其它服务器只读，不自行刷新
