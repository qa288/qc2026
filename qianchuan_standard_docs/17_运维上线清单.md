# 17 运维上线清单

## 1. 基线

- 域名：`ss.tyos.cc`
- 部署：Docker Compose
- 反代与站点管理：`1Panel`
- 路由：`/login`、`/`、`/api/*`
- 后台可公网访问，但必须登录

## 2. 服务清单

- web
- worker
- scheduler
- postgres
- redis

补充：

- 反向代理、证书和站点入口由 `1Panel` 托管
- 应用编排仍为 `Docker Compose`

## 3. 持久化

必须持久化：

- PostgreSQL 数据卷
- Redis 数据卷
- `/app/data`
- 日志目录

## 4. Token

- 只允许主服务刷新 token
- PostgreSQL 为主真源
- `/app/data/qianchuan_latest_token.json` 为缓存副本
- 其它服务器只读 token-latest 或缓存文件

## 5. 通知

- 第一阶段只验证 OpenClaw -> Feishu
- 不要求 DingTalk 同步上线

## 6. 资源建议

- 试运行：`2C4G`
- 正式生产（1000 账户）：`8C16G` 起

## 7. 上线前检查

- 域名已解析
- HTTPS 生效
- `1Panel` 站点与反向代理配置已生效
- PostgreSQL / Redis 正常
- Web / Worker / Scheduler 正常
- OpenClaw 测试消息可达
- token 真源可持续刷新
- 备份策略已落地
