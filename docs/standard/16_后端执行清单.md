# 16 后端执行清单

## 1. 模块范围

后端第一阶段必须完成：

- 认证与会话
- Token 管理
- 官方接口适配
- 结构同步
- 快照与聚合
- 账户汇总降级
- 阈值告警
- OpenClaw 通知

## 2. 必接官方链路

- auth_code 换 token
- refresh_token 刷新
- 子账户列表
- 计划列表
- 计划详情
- 商品
- 素材
- 自定义主题数据

## 3. 必须实现的系统规则

- Token 单点刷新
- 账户汇总降级
- 单账户失败不影响全局
- 单计划失败不影响其它计划
- 保留 request_id 日志

## 4. 持久化要求

- PostgreSQL 保存结构化数据与 token 真源
- `/app/data/qianchuan_latest_token.json` 保存 token 缓存副本
- Raw JSON 和导出文件保存在持久化卷

## 5. 通知要求

- 第一阶段只做阈值告警
- 走 OpenClaw -> Feishu
- 业务层不写死 Feishu / DingTalk 差异
