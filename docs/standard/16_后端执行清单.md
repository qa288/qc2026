# 16 后端执行清单

## 1. 模块范围

后端第一阶段必须完成：

- 认证与会话
- 角色与账户授权
- Token 管理
- 官方接口适配
- 结构同步
- 快照与聚合
- 账户汇总降级
- 归属人 / 关键词 / 人工修正
- 阈值告警
- 通知适配层与渠道配置

## 2. 必接官方链路

- auth_code 换 token
- refresh_token 刷新
- 子账户列表
- 计划列表
- 计划详情
- 商品
- 素材
- 自定义主题数据

## 3. 必须实现的系统规则

- Token 单点刷新
- 账户汇总降级
- 单账户失败不影响全局
- 单计划失败不影响其它计划
- 保留 request_id 日志

## 4. 持久化要求

- PostgreSQL 保存结构化数据与 token 真源
- `/app/data/qianchuan_latest_token.json` 保存 token 缓存副本
- Raw JSON 和导出文件保存在持久化卷

## 5. 通知要求

- 第一阶段只做阈值告警
- 支持余额 / 共享钱包 / 消耗 / 爆单规则
- 业务层不写死 Feishu / DingTalk / 其他渠道差异
- 通知发送可按开关后置，规则配置、触发记录和适配层必须先完成
