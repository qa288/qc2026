## 2. `docker-compose.yml` 示例

```yaml
version: "3.9"

services:
  web:
    build:
      context: .
      dockerfile: dashboard/Dockerfile
    env_file:
      - .env
    depends_on:
      - postgres
      - redis
    volumes:
      - ./data:/app/data

  worker:
    build:
      context: .
      dockerfile: dashboard/Dockerfile
    command: celery -A app.celery_app worker -l info
    env_file:
      - .env
    depends_on:
      - postgres
      - redis
    volumes:
      - ./data:/app/data

  scheduler:
    build:
      context: .
      dockerfile: dashboard/Dockerfile
    command: celery -A app.celery_app beat -l info
    env_file:
      - .env
    depends_on:
      - postgres
      - redis
    volumes:
      - ./data:/app/data

  postgres:
    image: postgres:17
    environment:
      POSTGRES_DB: ${POSTGRES_DB}
      POSTGRES_USER: ${POSTGRES_USER}
      POSTGRES_PASSWORD: ${POSTGRES_PASSWORD}
    volumes:
      - pgdata:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    command: redis-server --appendonly yes
    volumes:
      - redisdata:/data

volumes:
  pgdata:
  redisdata:
```

## 3. 1Panel 反向代理建议

在 `1Panel` 中建议配置：

- 站点域名：`ss.tyos.cc`
- 反向代理目标：`127.0.0.1:9898` 或对应 Docker 暴露端口
- 开启 HTTPS
- 打开 WebSocket 支持
- 保留 `Host`、`X-Forwarded-For`、`X-Forwarded-Proto`

如不使用 `1Panel`，也可以改用 Nginx / Caddy 实现同等反代。
