# 08 测试验收与上线清单

## 1. 测试范围

第一阶段测试范围：

- 公开页、登录与权限
- 主看板
- 账户 / 计划 / 商品 / 员工 / 素材数据
- 时间范围与自定义时间段
- 账户汇总降级
- 阈值告警
- 通知配置结构
- Token 刷新与共享
- 部署与回滚

---

## 2. 关键验收项

### 2.1 登录与权限

- 匿名用户可访问公开页
- 管理员可登录
- 运营可登录
- 未登录无法访问内部数据接口
- 运营与管理员同页，但数据范围受账户授权限制

### 2.2 页面与交互

- 总览页可正常展示
- 账户、计划、素材等榜单支持排序
- 自动刷新后排序不丢失
- 自定义时间段可用

### 2.3 数据链路

- 子账户同步成功
- 计划同步成功
- 素材同步成功
- 官方账户汇总异常时可降级展示

### 2.4 告警

- 阈值规则可新增、启停、删除
- 余额 / 共享钱包 / 消耗 / 爆单规则可配置
- 告警命中后可生成记录
- 去重和恢复可追踪

### 2.5 Token

- auth_code 换 token 可用
- refresh_token 刷新可用
- 最新 token 可被其它服务读取

---

## 3. 非功能验收

### 3.1 性能

- 页面查询优先读本地数据
- 高频榜单在合理时间内返回

### 3.2 稳定性

- 同步失败不导致服务整体不可用
- 单账户失败不影响全局

### 3.3 安全基础

- 后台接口必须登录
- Token 不暴露到普通页面接口
- 敏感配置不出现在前端

---

## 4. 上线前检查

- 域名 `qc.tyos.cc` 已解析
- HTTPS 生效
- `1Panel` 反向代理和站点配置已生效
- Docker Compose 可完整启动
- PostgreSQL / Redis 可连通
- 最新 token 可刷新和读取
- 数据卷、日志卷已持久化

---

## 5. 明确不再验收的范围

第一阶段不验收：

- 大屏模式
- 定时简报
- 日报 / 周报
